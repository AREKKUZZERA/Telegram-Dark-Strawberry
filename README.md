# 🍓 Telegram Dark Strawberry

Dark pink-themed Telegram Desktop theme with a soft strawberry aesthetic.

## ✨ Features
- Dark UI with pink accents
- Soft contrast for комфортного использования
- Styled chat bubbles, buttons и scrollbars
- Consistent color palette

## 📦 Installation

1. Download `.tdesktop-theme` file
2. Open Telegram Desktop
3. Go to Settings → Chat Settings
4. Click "Choose from file"
5. Select the theme
